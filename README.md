## 一款给velaOS的水平仪

## 快速上手
### 1. 开发

``` bash
npm install
npm run start
```

### 2. 构建

``` bash
npm run build
npm run release
```

### 3. 调试

``` bash
npm run watch
```
### 4. 代码规范化配置
代码规范化可以帮助开发者在git commit前进行代码校验、格式化、commit信息校验

使用前提：必须先关联git

macOS or Linux
``` bash
sh husky.sh
```

windows
``` powershell
./husky.sh
```
